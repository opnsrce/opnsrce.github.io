If you haven&#8217;t already, you should check out <a href="http://www.placehold.it" target="_blank">Placehold.it</a>. It&#8217;s a service that lets you dynamically pull in a placeholder image for use in working mockups and demos and it&#8217;s dead-simple to use. To pull in an image, you just reference the placehold.it url in an image tag:

<pre class="brush: xml; title: ; notranslate" title="">&lt;img src = "http://placehold.it/350x150"&gt;
</pre>

The above code pulls in a 350&#215;150 pixel image:

<img src = "http://placehold.it/350x150"/>

Need something with a little more pizzaz? There&#8217;s a <a href="http://placekitten.com/" target="_blank">version that uses kittens</a>:

<img src = "http://placekitten.com/350/150"/>
