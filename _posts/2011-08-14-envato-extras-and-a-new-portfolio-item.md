Recently, Envato announced a new website: <a href="http://extras.envato.com" target="_blank">Envato Extras</a>. The purpose of this site is to showcase applications, tools and utilities that people have built using the Envato API. A while back, I created a simple mobile app using Sencha Touch that used the Envato API to grab popular items from the Audio Jungle website and display them in a mobile friendly list. You can <a href="http://extras.envato.com/web-apps/audio-jungle-most-popular-items-app/" target="_blank">check out the application</a> on the Envato Extras website or you can head over to my portfolio and view the item there.
