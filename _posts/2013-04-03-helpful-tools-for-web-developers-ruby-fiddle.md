Recently, I talked about my [desire to learn more Ruby](http://www.levihackwith.com/learning-ruby-through-ruby-koans-online) to help round out my skill set. Well, if you&#8217;ve ever taken up the task of learning a new language, you know that the most difficult part can be getting the proper development environment up and running, especially if you&#8217;re trying to run an open source language on a Windows platform. If you&#8217;re looking to start messing around with Ruby with out having to install anything, I recommend you check out [Ruby Fiddle](http://rubyfiddle.com/). It&#8217;s basically <a href="http://jsfiddle.net/" target="_blank">JsFiddle</a> for Ruby developers and it&#8217;s a great way to both tinker with Ruby code and execute some of the examples in the <a href="http://koans.heroku.com/" target="_blank">Ruby Koans</a> project I mentioned in an earlier post.

As I&#8217;ve messed around with it, I have found a couple of weird issues. First, it appears that the &#8220;gistify&#8221; button doesn&#8217;t work as expected. Second, there&#8217;s no way for me to login and save the &#8220;fiddles&#8221; I make (JsFiddle allows you to do this). Hopefully these issues get fixed soon.
